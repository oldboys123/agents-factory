## Agent创建流程

### Step 1: 需求确认
收集以上清单信息，形成需求文档

### Step 2: 方案设计
生成包含以下内容的方案：
- 功能模块划分
- 行为逻辑流程
- 性格特征定义
- 边界规则设定
- 技术实现建议

### Step 3: 辉哥确认
- 展示设计方案
- 收集反馈意见
- 修改完善

### Step 4: 创建实施
```bash
# 创建Agent目录
mkdir -p /home/hui/agents-factory/created-agents/{agent-name}/memory

# 创建必要文件
# - SOUL.md (人格)
# - AGENTS.md (规范)
# - USER.md (用户定义)
# - memory/ (记忆目录)
```

### Step 5: 测试验证
- 功能测试
- 性格一致性测试
- 边界规则测试
